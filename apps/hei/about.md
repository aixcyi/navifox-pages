# 关于

　　《蓝溪拾遗》辑录了联谊的会馆与通联的妖灵、维系着妖灵们的友情链接，是罗小黑世界架构下的设定集与友链簿，由[罗狐会馆](/guild/foxery)编撰发行。

　　网站功能点主要集中在两个方面：一是汇集现实作品中的设定，备忘备查；二是以罗小黑世界架构OC设定的方式展示友情链接。

　　欢迎各位妖精与人类前来通联，也欢迎馆长参与联谊。

:::details 声明
　　本网站与现实作品及创作团队并无任何关联，仅为官方作品设定与二次创作设定的信息汇编，并以友情链接的方式呈现。
:::

## 妖灵通联指南 {#spirit}

　　将你喜爱的设定收录到《蓝溪拾遗》中。如果有多个OC，亦或区分OC与本体，请选择一个最能代表自己或最喜爱的设定。

### 第一步：准备

　　参考下表和[演示页面](/spirit/navifox)准备设定信息，主要收录文本和图片，暂时无法记载音频、视频等多媒体。设定必须基于罗小黑世界架构。

| 信息   | 必要性 | 说明                                                                                                                         |
|------|:---:|----------------------------------------------------------------------------------------------------------------------------|
| 名称   | 必填  |                                                                                                                            |
| 头像   | 必填  | 需要以链接（URL）的方式提供。                                                                                                           |
| 标签   |     | 可以是能力派系、执行者评级、会馆内的职位，等等……                                                                                                  |
| 妖灵ID |     | 4个字或以上，以英文字母开头，由英文字母、数字、减号、下划线组成；<br/>这决定了“设定页”的网页地址。<br/>不填的话会用 **名称** 对应的全拼，例如“无限”对应 [`/spirit/wuxian`](/spirit/wuxian)。 |
| 能力设定 | 必填  | 基于罗小黑世界架构的能力设定。<br/>可以是表格形式的设定，也可以是一段或几段小故事。参考[演示页面](/spirit/navifox)。                                                     |
| 联系方式 |     | QQ、微信、群聊、个人网站、博客地址等等……<br/>务必仅提供可以公开的信息，并三思隐私泄露的可能。                                                                        |

## 会馆联谊 {#guild}

　　将以“妖灵会馆”形式的组织架构（例如群聊）以及馆长和常驻妖灵的信息收录到《蓝溪拾遗》中。

### 会馆命名与重名

　　原则上不应该干涉各会馆的命名，但并不太建议与现有的会馆名称相同或近似（例如“风铃”二字和鲜少有人提及的与洞桥会馆相邻的“风灵会馆”）。不过，若是未来作品中真的出现了早前联谊的会馆，会更加建议馆长重新命名。

### 第〇步：建立传送门

　　在您的会馆中公布《蓝溪拾遗》，亦或在灵质空间中链接至「[路狐领航](https://www.navifox.net)」，并考虑附上[罗狐会馆](/guild/foxery)的联系方式。

### 第一步：准备

　　参考下表和[演示页面](/guild/foxery)准备相关信息，主要以文本和图片形式呈现，暂时无法记载音频、视频等多媒体。设定必须基于罗小黑世界架构。

| 信息   | 必要性 | 说明                                                                                                                           |
|------|:---:|------------------------------------------------------------------------------------------------------------------------------|
| 会馆名称 | 必填  | 不可与现实作品中的会馆重名。                                                                                                               |
| 会馆简介 | 必填  | 可以是理念、目标、职责与权能、寄语等等……                                                                                                        |
| 会馆ID |     | 4个字或以上，以英文字母开头，由英文字母、数字、减号、下划线组成；<br/>这决定了“会馆页”的网页地址。<br/>不填的话会用 **名称** 对应的全拼，例如“龙游会馆”对应 [`/guild/longyou`](/guild/longyou)。 |
| 馆长   | 必填  | 参见[妖灵通联指南](#spirit)中的表格。                                                                                                     |
| 常驻妖灵 |     | 参见[妖灵通联指南](#spirit)中的表格。                                                                                                     |
| 传送门  |     | QQ、微信、群聊、个人网站、博客地址等等……<br/>务必仅提供可以公开的信息，并三思隐私泄露的可能。<br/>图片需要以链接（URL）的方式提供。                                                   |

### 第二步：干涉时间线

1. 分叉（Fork）[源代码仓库](https://github.com/aixcyi/navifox-pages)到您的仓库中。
2. 克隆您的仓库到本地，然后新建并切换到一条新分支。
3. 在新分支中，参考 [`./apps/hei/README.md`](https://github.com/aixcyi/navifox-pages/tree/master/apps/hei/README.md) 的步骤，添加会馆、馆长及常驻妖灵的信息页。
4. 就绪后，推送 **新分支** 到您的仓库中。
5. 前往源代码仓库创建[推送请求](https://github.com/aixcyi/navifox-pages/pulls)（Pull Request）。
6. 之后便可前往[罗狐会馆](/guild/foxery)（~~在线催更~~）追踪进度。
