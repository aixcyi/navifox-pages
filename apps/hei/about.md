# 关于

:::details 声明
　　本网站与现实作品及创作团队并无任何关联，仅为官方作品设定与二次创作设定的信息汇编，并以友情链接的方式呈现。
:::

　　《蓝溪拾遗》辑录了联谊的会馆与通联的妖灵、维系着妖灵们的友情链接，是罗小黑世界观下的设定集与友链簿，由罗狐会馆编撰发行；欢迎各位妖精与人类前来通联，也欢迎馆长参与联谊。

## 妖灵通联指南 {#spirit}

　　参考[演示页面](/spirit/navifox)和下表准备OC设定信息，主要包括文本和图片，暂时无法接收音频、视频等多媒体，然后在源代码仓库中提出[申请](https://github.com/aixcyi/navifox-pages/issues)（Issue）；如果希望能自定义OC设定页面，可以前往源代码仓库提交[推送请求](https://github.com/aixcyi/navifox-pages/pulls)（Pull Request）。之后，可以前往[罗狐会馆](/guild/foxery)~~在线催更~~。

> [!NOTE]
> 所有图片都需要以链接（URL）的方式提供。

| 信息   | 必要性 | 说明                                                                     |
|------|:---:|------------------------------------------------------------------------|
| 名称   | 必填  |                                                                        |
| 头像   | 必填  |                                                                        |
| 标签   |     | 可以是能力派系、执行者评级、会馆内的职位，等等……                                              |
| 英文ID |     | 决定信息页的链接地址，不填就用 **名称** 对应的全拼，例如“木头”对应 `/spirit/mutou`。                 |
| 能力设定 | 必填  | 基于罗小黑世界架构的能力设定。<br/>可以是表格形式的设定，也可以是一段或几段小故事。参考[演示页面](/spirit/navifox)。 |
| 联系方式 |     | QQ、微信、群聊、个人网站、博客地址等等……<br/>务必仅提供可以公开的信息。                               |

## 会馆联谊 {#guild}

　　参考[演示页面](/guild/foxery)和下表准备相关信息，主要以文本和图片形式呈现，然后前往源代码仓库提交[推送请求](https://github.com/aixcyi/navifox-pages/pulls)（Pull Request）。之后便可前往[罗狐会馆](/guild/foxery)~~在线催更~~。

> [!NOTE]
> 所有图片都需要以链接（URL）的方式提供。

| 信息     | 必要性 | 说明                                                          |
|--------|:---:|-------------------------------------------------------------|
| 会馆名称   | 必填  | 不能与现实作品中的会馆重名。                                              |
| 会馆简介   | 必填  | 可以是理念、目标、职责与权能、寄语等等……                                       |
| 馆长     | 必填  | 参见[妖灵通联指南](#spirit)中的表格。                                    |
| 常驻妖灵   |     | 参见[妖灵通联指南](#spirit)中的表格。                                    |
| 传送门    |     | QQ、微信、群聊、个人网站、博客地址等等……<br/>务必仅提供可以公开的信息。                    |
